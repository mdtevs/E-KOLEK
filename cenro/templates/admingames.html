How It Works
  • Set individual cooldown periods for Quiz and Drag & Drop games
  • The Default cooldown applies to all games that don't have a specific configuration
  • Maximum cooldown is 168 hours (1 week)
  • Changes take effect immediately for new game sessions
  • Users in the mobile app will see the updated cooldown times
